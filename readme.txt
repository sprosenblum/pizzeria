/*========= About Theme =========*/

Theme Name: Pizzeria
Version: 1.0
Tested up to: WordPress 5.9
Author: Sarah Rosenblum 

== Description ==
Pizza Truck website with menu, pricing, location and hours.
Custom post types: pizzas, sauces, toppings, sides, locations, contact, our story.
Custom fields: map (for location), price (for pizzas and sides).

/*========= Theme Features =========*/

* Bootstrap 5.1
* Responsive design
* SEO friendly
* Responsive Menu
* Cross-browser compatibility
* Font Awesome icons

/*========= Documentation =========*/

Required plugins: Advanced Custom Fields by Delicious Brains and Custom Post Type UI by WebDevStudios.
